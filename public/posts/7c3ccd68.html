简述

昨天我因为要回老家，所以我想用手机写博客，
于是就可以搭建起来，搭建手机的环境其实很简单，
需要下载一个和一个termux

安装 Nodejs

安装长期稳定版本

pkg install nodejs-lts

或者你想要v14.0运行这个

pkg install nodejs

检查安装情况

node -V npm -V

安装Hexo

npm install hexo-cli -g
hexo -v
